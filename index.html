About the Project

Mapping the Hollywood Collaboration Network

Using the TMDB API, we collect popular films from 2010–2025, filtered by vote count and release window. From each movie's cast list we take the top eight credited actors.

An undirected, weighted graph is constructed where nodes are actors and edge weights count shared film appearances. Louvain community detection then partitions this graph into clusters of collaborating actors.

By linking each community to its movie plot summaries and running TF-IDF analysis, we test whether actor clusters also share thematic language — bridging network topology and natural language processing.
